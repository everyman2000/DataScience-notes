## Entering input
```R
x <- 1
print(x)

y <- 1:20 #creates an integer range
print(y)

z <- y[5] # subsetting
print(z)
```

```R
msg <- "hello"
```

`#` is used to comment. R supports only single line comments.

## Evaluation
```R
x <- 5
x    # auto-printing occurs
print(x) #explicit printing
```

The `[1]` indicates first element of vector.
